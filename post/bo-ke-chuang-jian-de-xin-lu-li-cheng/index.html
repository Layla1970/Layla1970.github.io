创建博客的心路历程

结庐人境

使用 云服务器 + Typecho

花了25元租了一年的一核 1G 的轻量服务器，这个大小对我来说已经够用了。
使用 Typecho 搭建博客的过程也非常简单。

  • 在服务器上安装宝塔面板，
  • 把 Typecho 下在解压到服务器中
  • 打开 index 页面按照步骤安装就可以了
💜主题

选择了DUX主题，刚开始看还挺好看的，后来越是看，觉得哪哪都不太好。
主题使用 PHP 写的，按照自己的喜好魔改了许多；改起来很方便，在宝塔面板中双击文件，弹出在线编译器，保存之后在页面上直接预览。

🈚️备案

由于是使用的国内服务器，网站上线必须备案。
备案真的是一个非常麻烦的过程，备案信息来回改了许多次，接入商的审核大概 2 个工作日，僵持了半个月，最终以备案被驳回告终。

🤡 记录一下备案失败的原因

  • 一开始是选择在江苏省备案，也耐下心性都所有信息都填写好，有两个文件需要署名（备案信息承诺书、备案信息真实性告知书），由于手机号码的归属地不在江苏，故换了一个省份备案。

  • 选择在山东省备案，我的手机号码的归属地是在山东省。山东省备案的个人网站信息驳回又做修改，改完后觉得天衣无缝，信誓旦旦的提交审核，但需要提供在山东省的居住证明，没有故驳回。

  • 填写的备案信息网站信息要求和江苏省不能说完全一样，至少毫不相关。比如江苏省的备案要求网站名称填写 “XXX的个人网站”或者“XXX的个人博客”，而山东省的个人网站备案不允许网站名称带有“网站”、“博客”等字眼。

使用  Github Page + Gridea

花了一块钱买的域名 huluu.xyz 也没有浪费，直接解析到 Github Page 上就可以了。

因为在 github 上托管，也就不需要麻烦的备案手续了，花了半天的时间把博客创建好，选了一个自己喜欢的主题，就可以开始使用了。

😣遇到的问题

  • xxx.github.io 打开一直在 404 ，原因是仓库名需要和用户名相同。
  • Gridea 同步失败，这个好像没什么办法，多试几次总会成功的；有时虽然提示同步失败了，但实际同步成功了。
